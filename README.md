💳 Credit-Card-Spending-Analysis---Power-BI-Dashboard
Interactive Power BI dashboard analyzing credit card transactions, spending patterns, customer behaviour, and category insights.

📊  Project Overview
This project presents an interactive Power BI dashboard analyzing credit card transaction data.  
The dashboard provides insights into customer spending behavior, transaction patterns, and category-based spending.

🔍 Key Insights

The dashboard provides analysis of:

- Total spending across transactions
- Number of credit card transactions
- Average transaction value
- Spending distribution by category
- Monthly spending trends
- Customer spending patterns

✨ Features
- Interactive Power BI visuals
- Category spending analysis
- Monthly transaction trend analysis
- Customer spending insights
- Clean dashboard design
